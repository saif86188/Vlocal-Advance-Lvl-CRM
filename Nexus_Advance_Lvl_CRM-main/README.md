# Nexus Apex — Intelligence CRM & Campaign Command Center

Next-generation administrative and client ecosystem built with **Next.js 15**, **React Server Components**, and the **Nexus Apex v4.0** framework.

## 🏗️ Project Architecture (NEXUS APEX v4.0)

This project follows a **Domain-Driven Modular** structure for maximum scalability:

- **`src/app/`**: Next.js App Router (Routing and Layouts).
- **`src/modules/`**: Feature-specific logic (Admin/Client domains).
- **`src/components/shared/`**: Global UI component library.
- **`src/styles/`**: Centralized design system and Tailwind config.
- **`src/lib/`**: Infrastructure, API clients, and utilities.

---

## 🚀 Quick Start

Ensure you have [Node.js](https://nodejs.org/) installed, then execute:

```powershell
# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:5174`.

---

## 💎 Core Features

- **Intelligence Feed**: Real-time notifications and strategy validation.
- **Campaign Hub**: Advanced workspace for managing multi-channel initiatives.
- **Command Center**: Comprehensive identity and profile management.
- **Engine-Driven UI**: Powered by the Nexus Apex design system (OKLCH, modular motion).
